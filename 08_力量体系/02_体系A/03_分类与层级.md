# 体系A — 分类与层级

Ξ 魔法无法凭空产生能量，也难以无准备地即时生效。其核心是依托特定介质构建能量转化通道、提升转化效率。技术体系分为三个层次，**每一层本质上都是对 n 的维度定义的一次拓展**——从纯物理振动到信息编码，再到神经通路直驱。

---

## 三层技术体系的底层模型

三层技术体系的本质，是"**n 载体的形态升级**"与"**复杂度控制维度的提升**"：

| 层级 | 名称 | n 载体形态 | 固有复杂度 | 主要特征 |
|---|---|---|---|---|
| **第一层** | 表层工具应用 | 人工机械振子 | n≈200-400 | 最安全，最普及，最受限 |
| **第二层** | 意念接口控制 | 仪器振子 + 人体发声器官（双 n 点系统） | n≈400-700 | 引入非物理尺度，生效维度从机械能拓展到形态控制 |
| **第三层** | 内源自发施法 | 大脑神经集群直接作为 n 点载体 | n≈850（接近 o 点） | 编码自由度最高，但极易触发负反馈与结构解离；不需要外部设备 |

每一层的 n 载体形态升级，都意味着可以利用更复杂的编码维度。但代价也随之升级——复杂度越高，解离风险越大。

## 第一层：表层工具应用

魔法的基础形态。依靠预制道具与能源介质实现能量转化，是最普遍的民用与工业应用形式。

### 技术原理

以人工机械振子作为 n 点载体，煤炭、水力、风力等外部能源作为输入，推动振子做小振幅高频往复运动。

振子的固有复杂度（n≈200-400）远低于 o 基准点（1000），正反馈循环稳定，复杂度解离速度极慢——一台标准工业振子的典型寿命在正常使用下可达数十年。

运动过程的复杂度低，散失率小，溢出的可用能量占比虽然不高，但因外部能量输入充足，绝对输出可观。

### 典型应用场景

| 设备类型 | 原理/用途 | 说明 |
|---|---|---|
| Ξ 锻锤 | 利用蓄势弹簧效应将旋转动能放大为定向冲击力 | 效率远超纯机械锻锤 |
| Ξ 加热器 | 通过场态热辐射实现材料的精准局部加热 | 工业与民两用 |
| Ξ 传讯器 | 利用微量溢出信号进行远距离信息传输 | 帝国远程通讯的骨干网络 |
| Ξ 泵 | 矿井排水和农田灌溉 | 最常见的民用 Ξ 工具 |
| Ξ 切割器 | 石材和金属加工 | 高精度工业应用 |

## 第二层：意念接口控制

技术积累于帝国时期，于帝国末期成型。该层利用人体 Ξ 场发声器官的调控能力，配合精密编码的仪器接口，实现用人的意念直接控制 Ξ 能量的输出与作用形态。属于高精尖技术领域。

### 技术原理

形成"**仪器主振子 + 人体发声器官**"的双 n 点协同系统：

- 仪器主振子负责实际的能量输出——它仍是机械振子，按照材料的物理规律工作。
- 人体副发声器官从信息维度修改 n 点形态定义——操作者发出的不是机械振动指令，而是编码信号。
- 转译接口接收人体信号和实时 f 测量值，通过仿生通用编码引擎计算驱动信号，指挥主振子产生等效输出。

这一层引入了非物理尺度的 n 定义：信息编码本身成为 n 形态的一部分，使得生效维度从机械能拓展到形态控制。系统复杂度升高，散失率相应提升。

### 工作流程

1. 操作者用人类自然语言逻辑（副发声器官的自然振动模式）表达意图。
2. 接口内的标准 Ξ 源校准模块实时测量当前 f 曲线参数。
3. 基于仿生通用编码引擎计算驱动信号。
4. 驱动仪器主振子执行精密振动。
5. 振子输出经过 f 曲线的传播卷积，在目标处产生与操作者意图匹配的 Ξ 效应。

操作者不需要知道 f 值是多少，不需要知道振子用什么材料，不需要学习"机器语"。他用人类的方式表达意图，接口负责翻译。转译接口内置的编码引擎会自主处理人类自然语言与当前 f 环境下最优机器语言之间的转译。

### 第二层技术的普及性特征

第二层技术的普及进程与帝国解体同步发生：

- 帝国末期，中央控制力减弱，原本被垄断在沿海研究社团和皇家工坊的转译技术随人才流失扩散至列国。
- 战国列国竞相建立自己的 Ξ 场研究机构，试图在接口技术上取得独立优势。
- "解码自然生物 Ξ 场编码"成为显学——不同国家组织远征队寻找新的 Ξ 场生物，试图发现更多的编码范本。
- **仿生编程生态**诞生：每个国家掌握着不同的生物解码库。有的从沙漠生物的解码中获得了耐高温环境的编码模块，有的从深海生物中提取了高压环境下的自适应算法，有的专注于飞禽类生物的远距离信号编码。
- 设备**互不兼容**：不是硬件差异，而是编码语言不通。缴获敌方设备后需要专门的反编译团队才能使用。
- 民间社团大量涌现：与帝国时期的严格管控不同，战国列国为了在技术上取得优势，对民间研究的限制相对宽松。各地的秘密社团、学院、行会都投入了 Ξ 场编码研究，形成了与官方并行——有时竞争、有时合作——的技术生态。

## 第三层：内源自发施法

狭义上"唯心感"最强的魔法形态。直至战国中后期才发展成熟，稳定传承体系的建立是该纪元与编程时代的分界标志。

### 技术原理

大脑神经集群直接作为 n 点载体，以人体生物能（ATP → 离子梯度 → 神经电位振动）作为外部输入，通过神经通路的兴奋-抑制循环产生内源 Ξ 信号。

大脑固有复杂度（850）接近 o 基准阈值（1000），编码自由度在三层技术中最高，但极易触发负反馈与结构解离。施法者需同时完成"维持神经复杂度"与"调控 Ξ 场输出"两层控制，才能实现对自源 Ξ 能量的编码与应用。

这是三层技术中唯一不需要外部设备的形态，也是最危险的形态。

## 文明演化史的双线驱动

Ξ 技术发展史并非单纯的文明迭代，也非单一 f 周期决定。其底层存在两条独立而交错的驱动线：

### 线 A：f 的周期律（物理环境线）

f 中心在数轴上的持续漂移，由多尺度干涉叠加混沌微扰构成。它决定了 Ξ 场通讯和 Ξ 效应在物理上是否可行、以及效率的高低。

- f 在人体范围内时：Ξ 通讯和第一层工具高效可用。
- f 远离时：必须依赖替代方案——声带语言、可调谐机构、转译接口、最终是大脑本身。

### 线 B：组织复杂度（文明内部线）

群体的规模、分工的精细度、代际知识传递的需求。这是一条**不可逆的单向箭头**。

组织复杂度越过阈值后，声带语言成为社会运转的刚需——即使 f 回归、Ξ 通讯恢复，也无法替代它。同样地，组织复杂度达到帝国规模时，人类选择把 Ξ 变成工业工具而非回归本能。战国规模下，民间研究和国家战略需求共同推动了第二、三层的成熟。

### 两条线的交错节点

上述两条线（f 周期律 × 组织复杂度）在不同历史阶段以不同方式交错，推动技术体系逐层跃迁。完整的七纪元划分与每条线的具体作用，见 [纪元划分](../../03_历史与年表/02_纪元划分.md) 的终章"重合的代价"。
