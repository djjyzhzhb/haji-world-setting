# 体系A — 原理与来源

本体系的核心不是一种超自然的"力量"，而是一套以**复杂度-效率相空间**为底层语言的物理描述。所有"魔法效应"均为这套描述下的自然现象——其触发依赖外部能量输入，其效率依赖一组会随时间漂移的环境参数，其代价是载体复杂度的不可逆耗散。

---

## 一、基础公理假设

整个体系在一个抽象的一维数轴上展开。该数轴并非物理空间的坐标，而是"复杂度-效率相空间"中的一条投影轴。

### 1. o 基准点

数轴上的基准参考点位，位于 **n=1000** 处。

o 点是宇宙固有的复杂度锚点。其位置在宇宙尺度下会发生极其缓慢的漂移，但在单个稳定恒星系的主要影响范围内可视为恒定。o 点是整个体系的**负反馈核心标尺**：载体复杂度越接近 o 点，Ξ 推动的边际收益越低，解离效应越剧烈。从底层逻辑看，o 基准点的存在必要性可被弱化（其极限可视为无穷大），但在人类文明所及的尺度上，它是一个实用且精确的工作假设。

物理意义上，o 点可被理解为"局部宇宙复杂度基态"——该恒星系内所有物质和能量系统的复杂度参考零点。

### 2. 效率分布函数 f

数轴上方存在一条对称钟型曲线（椭球分布族），曲线高度对应 Ξ 效应的作用效率。f 中心的位置与 n 值在同一个数轴上，两者的相对距离决定了效率：**n 越接近 f 中心，Ξ 效应的效率越高**。

效率函数 f 具有三个关键属性：

| 属性 | 决定因素 | 说明 |
|---|---|---|
| **曲线陡峭度** | 载体所处环境 | 不同地理区域、海拔、季节的温度气压变化影响陡峭度。环境越稳定，曲线越尖锐；环境越动荡，曲线越扁平。 |
| **效率峰值** | 载体自身复杂度 | 当载体复杂度低于 o 点时，峰值正比于复杂度；当超过 o 点时，峰值反比于复杂度。 |
| **周期性平移** | 多尺度天文周期 + 混沌微扰 | 曲线沿数轴随时间缓慢移动。星系级周期（~5000 时间单位）、恒星系级周期（~1500 时间单位）、局部扰动（~300 时间单位）叠加产生节律，如同气候系统。 |

### 3. 运动变量 n

数轴上的可变动状态点。n 代表 Ξ 场振动载体的工作位置——即载体振动的复杂度参数。

- 当 n 向 o 点趋近时，表现为 Ξ 的**推动效应释放**。
- 当 n 远离 o 点时，表现为 Ξ 的**积累状态**。

效率函数 f 的高度直接决定 n 产生 Ξ 效应的效率，n 自身的属性（影响范围、振幅、频率成分）会进一步参与效率调制。

### 4. 尺度定义规则

对 n 的取值维度与形态定义，将直接决定 Ξ 效应的**生效范围**——这是从第一层到第三层技术跃迁的理论基础：

| n 的定义尺度 | 对应技术层级 | 效果范围 |
|---|---|---|
| 纯物理尺度（机械振动频率、振幅） | 第一层工具 | Ξ 效应局限于物理空间内的能量转化 |
| 信息熵尺度 | 第二层意念编码 | 从能量拓展到形态控制 |
| 神经通路激活模式 | 第三层内源施法 | 大脑直驱，自由度最高但代价最烈 |

每一层本质上都是对 n 的维度定义的一次拓展。

## 二、能量转化与守恒规则

Ξ 效应的触发必须依赖**外部能量输入**——外部输入是推动 n 点状态变化的动力来源。Ξ 场本身不创造能量，仅作为能量的显性放大与效率增幅介质，同时对 n 的变化形态形成约束。

### 蓄势弹簧效应

这是 Ξ 能量生成的核心机制：

做小振幅高频往复运动的物体，在持续输入动能维持运动的区间内，会通过蓄势弹簧效应抽取周边及自身的部分**结构势能**，产生一股不受阻尼抵消的附加辅助动能（即 Ξ 动能）。这股附加动能可让物体仅需输入极小的维持力，就能保持稳定的高频振动幅度，同时持续向外辐射场态信息信号与微量热辐射。

"结构势能"的具体内涵因载体而异：

| 载体类型 | 结构势能的表现形式 |
|---|---|
| 机械振子 | 晶格应力、分子键的弹性势能 |
| 生物组织 | 细胞骨架的张力、膜电位的梯度能 |
| 神经系统 | 突触囊泡的储备、离子梯度的电化学势 |

蓄势弹簧效应与复杂度解离是同一微观机制的两个侧面：抽取结构势能即消耗结构有序性，放大与损耗互为表里。

### 能量去向

Ξ 能量的去向分为两部分：

**系统自耗部分**：大部分 Ξ 能量返还给运动系统本身，推动 n 点向 o 点的趋近运动，变化符合指数函数规律。占比由两个因素决定：

1. 运动过程的复杂度越高，推动效率越低，散失率越高。
2. 排除 Ξ 辅助后的 n 点自主变化量——即外部输入效率与系统自主运行效率的乘积。

系统无法在无外部输入、n 点无自主运动的情况下维持持续的 Ξ 效应。

**可利用溢出部分**：剩余的 Ξ 能量以可被直接利用的形式溢出。溢出形式取决于载体类型和编码方式——可以是定向的机械力（推动、切割）、场态热辐射（加热、熔融）、或信息信号（通讯、探测）。在系统与运动的复杂度较低时，溢出能量的量级极小，仅能作为噪声或信号使用。第一层工具的主要工程挑战，正在于如何最大化溢出部分并将其定向利用。

## 三、行星 f 场与局部 f 场

### 行星 f 场

前述效率分布函数 f 所描述的钟型曲线，其基本形态来源于行星级物理条件。行星 f 场依托于所在恒星系内稳定的天体运行关系与行星自身极大的质量，由此形成了稳定、广域、缓慢变化的基础效率分布。它的变化周期漫长，覆盖范围均匀，是所有局部 Ξ 场现象的**共同背景**。在设定中，"f 曲线"通常即指行星 f 场。

### 局部 f 场

任何持续产生 Ξ 效应的系统，若其内部复杂度足够高，并通过有意的放大与控制将 Ξ 输出维持在稳定状态，便可在自身周围形成一个叠加于行星 f 场之上的次级效率分布。

局部 f 场的共同特征：覆盖范围有限且与源强度相关，n 值位置和锐度取决于源系统的编码精度与复杂度特性。它们**不改变行星 f 曲线本身**，仅在局部复杂度空间中增加额外的效率结构。

按来源可分为两类：

| 类型 | 典型例子 | 说明 |
|---|---|---|
| **生物 f 场** | 大型长寿植物的生化代谢系统 | 内部持续的生化活动在复杂度空间中构成稳定的低频 Ξ 源，经演化筛选形成对自身无害但对某些病虫害构成干扰的局部场域。 |
| **人工 f 场** | f 塔、转译接口等文明主动构造的系统 | 工程基础与第二层转译技术同源，但应用方向不同：转译接口的目标是适应行星 f 场，人工 f 场的目标是在局部塑造一个叠加于行星 f 场之上的私有效率峰。 |

### 人类感知与发声的生物接口

副发声器官是演化赋予人类的**原生局部 f 场发生器**——气流振动 → 蓄势弹簧效应 → 尖锐可编码的次级 f 峰，其 n 值由意识决定的编码内容决定。感受器官则对 f 场的**变化率**而非绝对值敏感：脉冲式信号清晰可辨，宽缓平稳的场域（包括行星 f 场本身，以及缓慢部署的人工 f 场）被适应性过滤。

## 四、符号与数值总览

以下为体系 A 中所有核心符号的定义、物理意义、典型工作值与在人类文明史中的关键节点汇总。对数值的使用建议：**锚点值（粗体）应视为 txt 明确给定的约束；区间值（斜体）为 txt 描述推断的典型工作范围，仅作图示参考，读者可在 ±10% 的范围内调整而不改变形态趋势。**

### 4.1 核心符号定义表

| 符号 | 名称 | 在相空间中的含义 | 典型值或参考范围 | 决定因素 |
|---|---|---|---|---|
| **n** | 复杂度 / 运动变量 | Ψ 载体的工作位置（一维数轴上的可变动点） | *参见下方分层表* | 载体类型（机械/意念/神经）+ 编码精度 |
| **o** | o 基准点 / 复杂度基态 | 宇宙局部复杂度参考零点；n=o 是正负反馈的分界线 | **n=1000**（在单个稳定恒星系内视为恒定） | 星系级天体分布与质量-距离关系 |
| **f** | 效率分布函数 / 共振窗口 | 钟型曲线，在 n 轴上的位置（f 中心）决定不同 n 值载体的工作效率 | 人类文明史中 f 中心从 *≈250* 漂移至 *≈750* | 行星级物理条件（主导长期趋势）+ 局部人工 f 塔（叠加可利用峰） |
| **解离率** | 复杂度解离速度 | 载体因 Ψ 推动持续耗散自身结构势能的速率 | *与 (n/o) 呈正相关；n 越接近 o 风险越高* | 载体内在复杂度与环境陡峭度的组合 |
| **共振宽度** | 曲线陡峭度的倒数 | f 曲线半高宽，决定"能用的 n 值范围有多宽" | *稳定环境 ≈80–150；动荡环境 ≈250–400* | 地理/海拔/季节；局部人工 f 场可被工程调制 |

### 4.3 人类文明史中的 f 中心漂移轨迹

*此段与 [纪元划分](../../03_历史与年表/02_纪元划分.md) 中的文明分期一一对应。*

| 纪元 | 历史时期 | f 中心（示意） | 说明 |
|---|---|---|---|
| 本能时代 | 前文明期（人类尚无 Ξ 技术） | **≈250**（在人体自然感知范围内） | 人类 Ξ 感知器官（感受器官）最敏感的 f 区间 |
| 沉默时代 | 早期农业期 | **300** | f 开始缓慢偏离人体范围，语言复杂度上升 |
| 帝国早期 | 古代大一统帝国 | **300 → 450** | 固定调谐的第一代机械振子出现，以 f≈300 为主流 |
| 帝国中后期 | 技术成熟与扩张期 | **450 → 550** | 可调谐旋钮与合金钢材料工程追赶上 f 的漂移 |
| 战国时代 | 分裂与军事竞赛期 | **≈750** | f 塔 + 多相位联合调制；合成峰精准但不稳定 |
| 内源时代 | 第三层施法成为新支柱 | *继续向 o 方向推进（>750）* | 大脑成为最后可用的高 n 载体，代价也最高 |

> f 中心在 *≈250* 与 *≈750* 之间的漂移代表了本设定的核心剧情张力——每当文明工程追赶上一次漂移，f 曲线已经在新的位置等待。
