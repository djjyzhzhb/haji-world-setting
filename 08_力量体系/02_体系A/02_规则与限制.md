# 体系A — 规则与限制

本文件描述体系 A 中所有"不能做的事"——代价、边界、失效条件与内在矛盾。这些规则是体系设计的**负空间**，定义了力量的天花板。

---

## 一、三条核心约束

所有 Ξ 效应均受以下三条规则约束，从根源杜绝战力膨胀与永动机悖论。

### 规则 1：外部输入守恒

Ξ 场仅做效率放大与形态转化，**本身不创造能量**。

没有煤炭、水力、或人体 ATP——没有 Ξ 效应。系统无法在无外部输入、n 点无自主运动的情况下维持持续的 Ξ 效应。

### 规则 2：复杂度解离代价

所有 Ξ 推动都伴随载体复杂度的**不可逆耗散**，不存在无代价的永久施法。

| 载体类型 | 解离的具体表现 |
|---|---|
| 机械振子 | 金属疲劳、晶格位错累积、微裂纹扩展 |
| 生物组织 | 蛋白质变性、膜结构破坏 |
| 神经组织 | 突触连接断裂、髓鞘脱落、神经元凋亡 |

蓄势弹簧效应与复杂度解离是同一微观机制的两个侧面：抽取结构势能即消耗结构有序性，放大与损耗互为表里。

### 规则 3：o 点基准负反馈

载体复杂度超过阈值（n=1000）后，效率骤降、损耗暴增，边际收益随接近阈值持续递减。

具体机制：

- o 点自带一个基准复杂度系数，该系数与 Ξ 的推动速率负相关。持续受 Ξ 推动的系统，其复杂度会随推动时长逐步解离消耗。
- 若载体固有复杂度高于 o 点基准值，复杂度提升会带来负反馈：等效于 n 点远离 o 点，即便效率峰值随复杂度上升有所提高，整体也无法获得净收益，且解离效应会进一步加剧。
- 若载体固有复杂度低于 o 点基准值，则可形成正反馈循环。常规系统（第一层机械振子 n≈200-400，第二层双 n 点系统 n≈400-700）远低于 o 点，正反馈循环稳定。
- 正反馈仅为理想模型下的结论。实际存在大量客观限制：指数增长会在触及 o 点上限前，因 n 点自主变化量的客观上限而放缓。

## 二、安全施法窗口（第三层）

神经结构的解离速度与施法强度呈**超线性关系**——强度提高一倍，解离速度提高约三倍。

存在一个"安全施法窗口"：在此窗口内，大脑的自主修复机制可以基本抵消解离。超过窗口，损伤以不可逆的方式累积。

### 编码模式的外包机制

通过反复练习，施法者可以将某些固定的编码序列**嵌入小脑的自动化回路**，从而在施法时减少对脆弱脑区的直接调用。这是"安全神经活动路径"的生理基础——将常用编码外包给自动化系统，以降低主路径的复杂度压力。

## 三、个体偏差与统计安全（规则性陈述）

安全神经活动路径是**统计安全，而非绝对安全**。以下为事实层面的硬性边界，完整的训练困境、个体案例与传承哲学见 [传承体系与魔法社团](../../06_政治与势力/传承体系与魔法社团.md)。

- **个体差异不可消除**：人脑功能区的精确边界、神经束走向、侧化程度因人而异，无技术手段可对活体大脑精确成像。
- **偏差触发不可预测**：低阶编码中可能永远不暴露的细微偏差，在高阶/高压或新编码模式下可能突然触发，导致非预期脑区损伤。
- **训练与风险的根本矛盾**：降低未知偏差风险的唯一方法是延长低强度试探周期——这与快速晋升、快速产出的需求天然对立。
- **"最优"不可识别**：看似天赋异禀的快速晋升者，其神经结构与标准路径的匹配程度无法被验证；而缓慢、谨慎的存活者反而建立了对自身大脑的真实认知。

## 四、体系边界总结

| 限制类型 | 具体内容 | 影响范围 |
|---|---|---|
| **能量守恒** | 无外部输入则无 Ξ 效应 | 所有层级 |
| **复杂度解离** | 每一次施法都在消耗载体——机械或神经 | 所有层级；第三层代价最致命 |
| **o 点负反馈** | 复杂度越高效率越低、损耗越烈 | 第三层最受影响——大脑 n≈850 已接近 o 点 |
| **f 漂移** | 有效 f 窗口随时间变化，固定设计会失效 | 第一层最明显；第二层通过转译适应；第三层受大脑固定复杂度的限制 |
| **个体偏差** | 安全路径是统计概念，每个人的大脑是黑箱 | 第三层的核心风险，无法被技术完全消除 |
| **编码隔离** | 各国生物解码库不同，设备互不兼容 | 第二层的战略竞争维度 |
| **感知过滤** | 对变化率敏感而非绝对值——缓慢变化的场域不可被感知 | 战略层面：f 塔可隐秘部署 |
