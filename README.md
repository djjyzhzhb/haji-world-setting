# 哈吉语世界观设定集

> 本项目是**人造语言创制计划**（哈吉语）的子项目 —— 为人造语言提供文化背景与文明土壤。

## 项目性质

这是一个作为文学创作基础的**可运行世界观设定集**，而非单一线性故事。世界先于人物存在，人物是世界条件的产物。

## 核心美学原则

**广义现实主义（Mimesis）**：源自亚里士多德《诗学》—— 摹仿不是复制现实，而是揭示事物的内在本质和因果必然性。

- "合情合理的不可能" 优于 "不合情合理的可能"
- 每个事件必须有前因，每种社会结构必须有物质/历史条件支撑
- 所有超自然力量必须有边界和代价

## 当前阶段

**设定创作阶段** —— 不急于产出完整叙事，先构建世界的骨骼。
