# 哈吉语 192 元素汉字选字池

> **A 池**＝标准音译用字。按《外语译音表》风格收录，每个音节通常有 2~5 个常规选项。无特殊审美意图，纯表音。
> **B 池**＝风格化音译用字。读音必须对得上，但不关心原意，只看字面风骨。生僻字标 ※。
> **C 池**＝义译字。不看读音只看含义。仅兜底几个实在难音译的核心概念。

## 使用指南

```
A 池：标准音译，普通词汇主力。每个音节 2~5 个常规选项。
B 池：风格化音译，人名/地名/诗名时捞取。读音必须对得上元素音节。
C 池：义译，不管读音。仅少数核心概念动用。

翻译策略：
├── 普通词 → 全 A
├── 姓 → B 捞 1~2 字，来历感优先
│   例：mo(B猫) + gu(B古)
├── 名 → 70% A + 30% B 点睛
│   例：na(A娜) + xo(B笑) + mi(A米)
├── 称/小名 → B 为主，轻快即可
│   例：qo(B桥) + ko(A可)
├── 地名 → B 里找山水草木建筑感
├── 歌词 → A/B/C 自由，押韵优先
└── 术语 → A 为主，必要时动 C
```
