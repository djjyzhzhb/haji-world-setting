# 简单介绍一下哈吉语这一名字的由来吧

哈吉语是一门几乎所有词汇都以复合词的形式诞生的语言，haji 这一名称也不例外，它由两个基本语素 ha 与 ji 构成。

它们表示：**欲望** 与 **猎杀**

那么为何一个这么血腥的词汇会成为一门语言甚至一个民族的名字？这还需要从哈吉语的背景设定讲起：

千年、万年前，在一片名叫启音大陆(qikoükung)的土地上，诞生了一群使用着后人称之为启音语(qiko)的语言的人，它们拥有原始的自然信仰，认为天地万物乃是众神(naiding)的化身。

据不可考证的传说记载，启音语的语言极其纯粹，共8×24=192个绝不可再分的音节。一个音节乃是一位神明在世间的位置，乃是代表了一种元素(yü)，世间万物皆由元素构建，语言也不例外。

在轴心时代来临后，中层知识分子阶级掌握了文化话语权，面对当前语言上"礼崩乐坏"的场面，他们进行了彻底的大修正，根据现代读音，定义构建了标准的启音语——一个他们想象中的启音语。

而这份变革并不是统一完整的，更靠近启音语发源地的西部高原山脉相对保守的人们与其思想家，更乐于接受与推进这份"启音正音"的方案，而居于东侧少量平原的沿海地区的人们，他们自由实用得多，他们最终接受了启音的文化脉络，却抗拒了"正音"的改化，自称为法奇(faqi)人，意为"自主意志者"。

西部的上层为构建权力根基，强调其正统性，更积极推进了这份工程，并为全国确立了持有此文化的人们的称号——**haji**。

- **ha**：欲望，代表古启音人从自然的众神中取得了元素"启音"的历史
- **ji**：猎杀，代表最初的启音人受到了赐福的纯净的口腔

所以，**haji——受启音正音者**。

附：一个美妙的巧合是，在地球上同样有一个词汇名叫"哈吉/hajji"，它的含义是——完成朝圣者。
