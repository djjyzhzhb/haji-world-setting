# 世界一致性规则手册

## 一、绝对禁止事项

### 1.1 因果律破坏
- **禁止**：无前因的事件发生。任何事件必须有可追溯的起因。
- **禁止**：事件发生后无后果。每个行动必须产生与其规模匹配的连锁反应。
- **禁止**："因为剧情需要"式的强行巧合。

### 1.2 物质基础悬空
- **禁止**：超出现有技术水平的生产力。
- **禁止**：没有经济基础支撑的社会结构（如：没有农业盈余的庞大城市）。
- **禁止**：没有资源来源的军事力量（如：没有铁矿的钢铁军团）。

### 1.3 力量体系失控
- **禁止**：无代价的魔法/超自然力量。
- **禁止**：力量使用不产生社会后果（如：存在复活术的社会仍视死亡为终结）。
- **禁止**：力量持有者不受社会结构的约束。

### 1.4 文化逻辑断裂
- **禁止**：与生态环境矛盾的文化习俗（如：沙漠民族崇拜水神但铺张用水）。
- **禁止**：与历史记忆矛盾的社会行为（如：被侵略过的民族对侵略者毫无警惕）。

## 二、必须遵守的铁律

### 2.1 因果链原则
```
每个设定点的回答必须包含：
  起因 → 表现 → 后果 → 与其他设定的因果联系
```

### 2.2 物质条件优先原则
在解释任何社会现象时，优先考虑：
1. 地理环境 → 2. 经济基础 → 3. 社会结构 → 4. 文化表现 → 5. 个体心理

### 2.3 语言一致性原则
- 所有命名必须遵循 `12_附录与参考/01_命名规则总表.md` 中的文化/种族命名规范
- 同一概念在不同语境中必须使用统一术语（以 `_glossary.md` 为准）
- 对话中的语言风格必须与角色的社会阶层、教育背景匹配

### 2.4 时间一致性原则
- 所有事件时间标注必须符合 `01_世界核心/04_时间体系.md` 中的历法体系
- 角色年龄、事件间隔必须经得起时间线推演

## 三、风格边界

### 3.1 叙事语调
- 保持冷静、客观的"设定集"语调，不煽情、不夸张
- 描述性语言优先于评价性语言
- 避免现代口语和网络用语渗入叙事

### 3.2 描写密度
- 关键设定允许详细展开（如核心力量体系）
- 次要设定保持简洁，仅提供必要信息
- 留白是美德：不确定的细节可以标注"待定"而非强行填充

## 四、矛盾处理流程

当发现前后矛盾时：
1. 在 `13_因果律与一致性/06_矛盾记录与修正.md` 中记录矛盾
2. 分析矛盾的根本原因
3. 确定修正方案（修改较早的设定 or 修改较晚的设定）
4. 追溯所有受影响的设定文档并统一修正
5. 更新版本日志
