## 核心数字速记

- **n = 1000**：o 基准点，Ξ 效应安全上限。
- **n ≈ 200–400**：第一层机械振子有效区间。
- **n ≈ 400–700**：第二层转译接口有效区间。
- **n ≈ 850**：第三层大脑施法危险区间。
- **f 中心 ≈ 300**：帝国早期稳定位置，对应人体舒适度区。
- **f 中心 ≈ 450–550**：帝国中晚期，旋钮和材料工程承压。
- **f 中心 ≈ 750**：战国早中期，大脑可产生可测量 Ξ 信号。
- **f 中心 ≈ 850**：战国中后期，大脑直驱进入危险可行区。
- **死亡率 ≈ 70%**：第三层传承体系早期训练死亡率。

## 双线驱动框架

历史由两条独立线索交错形成：

1. **物理环境线（f 周期）**：f 中心在数轴上漂移，决定 Ξ 通讯与 Ξ 工具在物理上是否可行及效率。
2. **组织复杂度线（文明自身）**：群体规模、分工精细度、代际知识传递需求。不可逆。

交错节点：
- 语言独立：f 偏离触发，组织复杂度保留。
- 第一层工具化：f 回归 + 帝国规模需求。
- 第二层突破：f 漂移 + 战争/生产需求 + 神秘生物启发。
- 第三层诞生：f 逼近大脑 + 第二层神经控制经验。
- f 塔网络：施法者威胁 + 多塔联合调制技术。

详细论述见 [因果链图谱](../03_历史与年表/04_因果链图谱.md)。

## 常见误读澄清

| 误读 | 正解 |
|------|------|
| "Ξ 场是魔法能量" | Ξ 场是物理场，不创造能量，只放大转化效率。 |
| "f 周期是固定轮回" | f 周期是多尺度叠加 + 混沌微扰，非精确可预测。 |
| "f 回归会让语言消失" | 不会。组织复杂度不可逆。 |
| "第三层魔法师是最强战斗单位" | 直接战斗是低效浪费。魔法师的价值在于编码设计、训练他人与特殊任务。 |
| "f 塔能压制所有施法者" | 不能。它只能提高施法者的环境风险，迫使对方进行场探。 |
