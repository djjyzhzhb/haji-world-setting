# 一致性检查清单

> [AI关键] 每次生成内容后的自检工具。

## 生成后必查项

- [ ] 是否有前因？（事件是否有可追溯的起因？）
- [ ] 是否有后果？（事件是否产生了合理的连锁反应？）
- [ ] 物质条件是否支撑？（技术水平、资源、经济基础是否允许？）
- [ ] 力量体系是否越界？（是否有无代价的力量使用？）
- [ ] 术语是否统一？（是否使用了 `_glossary.md` 中的标准名称？）
- [ ] 时间是否一致？（是否符合 `01_世界核心/04_时间体系.md` 的历法？）
- [ ] 风格是否一致？（是否符合 `_style_guide.md` 的要求？）
- [ ] 有无矛盾？（是否与现有设定冲突？）
